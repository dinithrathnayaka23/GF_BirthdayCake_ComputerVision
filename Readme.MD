# 🎂 Virtual Birthday Cake - Computer Vision Web App ❤️

A fun and interactive virtual birthday cake experience built using **HTML, Tailwind CSS, JavaScript, Canvas API, MediaPipe Computer Vision, and Web Audio API**.

The idea behind this project is to create a virtual birthday celebration for someone special who is far away. The user can open the web application, see an animated birthday cake, play the Happy Birthday song, and blow into the microphone to extinguish the candles using computer vision and audio processing.

---

# ✨ Features

## 🎂 Interactive Virtual Cake

- Canvas-based birthday cake rendering
- Multiple candle flames
- Realistic flickering flame animation
- Candle extinguishing animation
- Smoke particle effects

---

## 💨 Blow Detection

Uses the browser microphone to detect blowing.
